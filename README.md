# Accessory Coach

Mobile-first app prototype for your CrossFit + Olympic lifting accessory program.

## What is included

- Today's accessory recommendation based on readiness and what class already included
- 5-day accessory program built around your lifting template
- Daily ankle and shoulder mobility routine
- 8-week tracker stored locally on the phone/browser
- Progress test tracking for squat, front squat, snatch, clean and jerk, double-unders, bar muscle-up, Grace, and heavy-WOD capacity
- Offline-capable PWA files after it is hosted

## How to test it locally

Open `index.html` in a browser.

## How to put it on your phone as an app

The simplest no-code method:

1. Create a free Netlify account.
2. Drag the whole `crossfit_accessory_app` folder into Netlify Drop.
3. Open the Netlify link on your phone.
4. On iPhone: Share button -> Add to Home Screen.
5. On Android/Chrome: menu -> Install app or Add to Home screen.

You can also host the same folder on GitHub Pages, Cloudflare Pages, or Vercel.

## Notes

Your logs are stored in the browser's local storage on that device. Use the Export button inside the Tracker tab to back up your data.
